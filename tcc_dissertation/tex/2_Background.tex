Biologically, the brain cell has needs to receive an excitation higher than it`s threshold to trigger an impulse, which is determined by their own characteristics.

Considering that, the authors claim that a neuron can be described as a logic block, executing logical function. Brain cells could be organized in nets that could describe any logical proposition. In 1950, with the question 'Can machines think?', Alan Turing starts his paper \cite{DBLP:journals/x/Turing50}, describing thinking abilities as problem-solving, decision-making and pattern recognition. By stating that machines could emulate human intelligence, it`s necessary to delimit what it is and how it can be measured. And that would be the parameter to determine whether or not machines had achieved human intelligence.


By comparing the human brain to digital computer, Turing suggests that it was not necessary to build a machine comparable to an adult human intelligence capacity, but one with capacity of apprenticeship. Suggesting that a "child machine" should be subject to "education" through rewards and punishments, analogous to how children learn. \gls{ML} is an area of knowledge that focus on developing algorithms that allows computer system to learn from data and improve their performance without being explicitly programmed, this mathematical relationship derived from the data is called model. \section{Neurons and Supervised Learning}

Frank Rosenblatt work \cite{rosenblatt1958perceptron} proposed in 1958 a model to explain how the human brain storages data and utilizes it for pattern recognition and behavior. He was compromissed in understand how information contained in storage influence recognition. Like Alan Turing \cite{DBLP:journals/x/Turing50}, Rosenblatt also belived in the application of positive or negative reinforcement to make the system learn. Not only that, but also expanded to his machine model the biological concept that similar stimuli should activate the same nets cells.

This probabilistic model called perceptron is divided in three layers: Sensory units that responds to impulses in all or nothing bases; Association units receives the transmitted impulses and, if the algebraic sum of those are greater than the unit threshold, it fires and generates the comebacks of some Response units. Each response unit has feedbacks connections that can be excitatory or inhibitory and are mutually exclusive. This means that, if a response unit is triggered during a stimulus, it will inhibit others from trigger to other similar ones.

\begin{equation}

With that, minimizing \gls{MSE} is equivalent to minimizing the average number of neuron error, which lead to the best set of weights. \section{Artificial Neural Networks and Gradient Descent}
\label{sec_neural_networks}
The perceptron \cite{rosenblatt1958perceptron} can be considered as the first published attempt to build a \gls{ANN} that could learn from examples. However, it was only a single layer aside from input and output. Later, in his book Principles of Neurodynamics \cite{Rosenblatt1963PRINCIPLESON}, the author explicitly discusses multilayer perceptron architectures, with multiple neurons organized in series. As in figure \ref{fig:adjustable neuron}, the output of a neuron is connect serves as input of the next one.

It was with David E. Rumelhart et al \cite{Rumelhart1986LearningRB} that an \gls{ANN} training procedure was shown useful in practice, because of the high cost of adjusting all weights of the neuron. Those extra layers between output and input were called hidden-layers, or hiddens units, and they introduce more difficult to find the learning rules, or the corrects weights, to reduce the difference between the actual and desired outputs. The input of a neuron in j layer is a linear function of the outputs ($y_i$) multiplied by their respective weights ($w_{ij}$).
\begin{equation} 
    x_j = \sum_{i} y_i w_{ij} 
\end{equation}
One important difference pointed in \cite{Rumelhart1986LearningRB} is each hidden-layer output is followed by a non-linear function, different than \gls{ADALINE}, which all layers had as output a linear-function. This occurs because series linear function only could be simply replaced by a single one, just adjusting the weight, which would limit the system capacity of learning.
For example, in binary classification, as explored in chapter \ref{chap:methodology}, is very common to use sigmoid function as the non-linear function of the neurons output:
\begin{equation}
     y_j = \frac{1}{1 + e^{-x_j}} \label{sigmoid_func}
\end{equation}
David appointed that the total error of an multi-layered \gls{ANN} could be, similar to perceptron, as the comparison of the desired output and the prediction
\begin{equation} 
    E = \frac{1}{2} \sum_{c} \sum_{j} (y_{j,c} - d_{j,c})^2 
\end{equation}
where ($c$) is an index over cases, ($j$) is index over outputs units and ($y$) the actual state of the output. The main point, as in Rosenblatt article \cite{Rosenblatt1963PRINCIPLESON}, is minimize ($E$). To achieve it, is suggested the use of \gls{GD} \cite{5a25e804-97fa-3be4-ac27-50be05d4f28d}, mathematical method to find local points of minima.
By calculating the partial derivatives of the error and applying calculus chain rule, it is possible how the change in the input or the weight of a layer ($j$) affects the error:
\begin{equation} 
    \frac{\partial E}{\partial x_j} = \frac{\partial E}{\partial y_j} y_j (1 - y_j) 
\end{equation} 

\begin{equation} 
    \frac{\partial E}{\partial w_{ij}} = \frac{\partial E}{\partial x_j} y_i 
\end{equation}

The derivates above propagates from the top layer back to the bottom one after, updating the weights of the \gls{ANN}:

\begin{equation} 
    \Delta w(t) = -\epsilon \frac{\partial E}{\partial w(t)} + \alpha \Delta w(t-1) 
\end{equation}

Where ($t$) where ¢ is incremented by 1 for each sweep through the whole set of input-output cases, and ($\alpha$) is an exponential decay factor between 0 and 1 that determines the contribution of the current gradient and earlier gradients to the weight change.

 This new method of learning with \gls{BP} and \gls{GD} made multilayer neural networks available in every-day use, overcoming perceptron limitation that could only solve linearly separable problems.

\subsection{Convolutional Neuron Networks}
\label{sec_cnn}
Kurt Hornik and Maxwell Stinchcombe and Halbert White \cite{HORNIK1989359} established hat standard multilayer feedforward networks are capable of approximating any measurable function to any degree of accuracy. Which implies that applications can reach desired results by an adequate learning process, numbers of hidden units or the quality of a deterministic relationship between input and target. 

In Y. LeCun paper \cite{6795724}, different from other previous results, was proposed an architecture with learning procedure directly fed with images, rather than feature vectors. The main goal to achieve was the image recognition from handwritten zip code digits from U.S Postal Service. Another contrast from other earliers work was the fact that all connections in the network were trained using \gls{BP}, where the first few layers of connections used to be handchosen constants implemented on a neural-network chip for the specific task. To storage all the complex information derived from an entire image instead of a vector and extract common features in pattern recognition, the weight share technique was apllied to extract local features and combine them to form higher order features.

The proposed network had three hidden-layers: two convolutional feature-extraction layers followed by one fully connected classification layer, which were later referenced as a \gls{CNN}, by the fact the function performed by a feature map can thus be interpreted as a nonlınear subsampled convolution with a 5 by 5 kernel, which is the set of connection weights used by the units in the feature map, as described in equation \ref{convolution}.

\begin{equation}
x_j^{\,l}
=
f\!\left(
\sum_{i \in M_j}
x_i^{\,l-1} * k_{ij}^{\,l}
+
b_j^{\,l}
\right)
\label{convolution}
\end{equation}

Yann LeCun et all \cite{articleDocReg} describes a \gls{CNN} as a network that combine three architectural: local receptive fields, shared weights, and spatial or temporal sub-sampling.  The local receptive fields can extract elementary visual features such
as oriented edges, end-points and corners. These features are combined by the subsequent layers in order to detect higher-order features, such as object detection. 

All the units in a feature map share the same set of weights and the same bias so they can detect the same feature at all possible locations on the input, while a convolutional layer is composed of several feature maps with differets weights vector, to multiple features be extracts in the same locations.

Furthermore, \gls{CNN} architecture owns a sub-sampling layers which performs a local averaging and a sub-sampling, reducing the resolution of the feature map. For example, the second hidden layer of LeNet-5 comprises six features maps, each unit computing the average of its four inputs, multipling it by a trainable coeficient and then adding a trainable bias, which the result is passed through a sigmoid function.

\begin{equation}
\sigma(x) = \frac{1}{1 + e^{-x}}
\label{sigmoid_function}
\end{equation}

\section{Evaluated Metrics}
\label{sec_metrics}
In classification models in tasks such as image and pattern recognition, it`s important to have a way of mesuare how well a \gls{DL} model is performing. The simplest case of classification is the binary classification, where there are only two classes a sample can be, normally labeled as Positive (P) or Negative (N). There`re total outcomes of four outcomes to a binary classifier \cite{metrics}:
\begin{itemize}
    \item \gls{TP}: Predictions where the model correctly predicted the sample to belongs to a "class A".
    \item \gls{TN}: Predictions where the model correctly predicted the sample to not belongs to a "class A".
    \item \gls{FP}: Predictions where the model incorrectly predicted the sample to belongs to "class A".
    \item \gls{FN}: Predictions where the model incorrectly predicted the sample to not belongs to "class A".
\end{itemize}

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.5\textwidth]{outcomes-of-classifier}
    \caption{Representation of the four outcomes of a classifier \cite{metrics}.}
\end{figure}

The most common way to visualize those outcomes is using the confusion matrix. It`s a table with four different combinations of predicted and actual value of labeled samples \cite{metrics}.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.5\textwidth]{confusion-matrix-explain}
    \caption{Confusion matrix for binary classification problems \cite{metrics}.}
\end{figure}

Considering the possibles outcomes of binary classification sample, there are different metrics, which evaluates different characteristics of the model performance that should be considered during training phase. As discussed in \ref{chap:methodology}, four of those metrics were used during this work, which will be explained next.

\subsection{Accuracy}
\label{sub_sec_accuracy}
Accuracy is the number of all corrects predictions by the total of the number of the dataset.
\begin{equation}
    \frac{TP + TN}{TP + FP + TN + FN}
\end{equation}

\subsection{Recall}
\label{sub_sec_recall}
Also called sensivity, recall is the number of correct positive predictions by the number of total positives. It can be interpreted as the model capacity to correctly identify positive instances.
\begin{equation}
    \frac{TP}{TP + FN}
\end{equation}

\subsection{Specificity}
\label{sub_sec_specificity}
Specifity is the number of correct false predictions by the number of total falses. It can be interpreted as the model capacity to correctly identify negative instances.
\begin{equation}
    \frac{TN}{TN + FP}
\end{equation}

\subsection{Precision}
\label{sub_sec_precision}
Precision is the number of correct positive predictions by the total number of positive predictions. It can be interpreted as the reliability of a correct positive prediction.
\begin{equation}
    \frac{TP}{TP + FP}
\end{equation}

\subsection{F-score}
\label{sub_sec_fscore}
F-score is a harmonic mean of precision and recall.
\begin{equation}
    \frac{2*Precision*Recall}{Precision*Recall} = \frac{2*TP}{2*TP + FP + FN}
\end{equation}
